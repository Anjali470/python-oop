# Python OOP Projects

This repository contains basic projects built using Object-Oriented Programming (OOP) in Python.

---

## Student Management System (OOP)

A student system using classes to manage data.

### Features
- Create student profiles
- Enroll in courses
- Add grades
- Calculate GPA and CGPA
- View student details and transcript

---

##  Banking System

A simple banking system using inheritance.

### Features
- Deposit and withdraw money
- Check account balance
- Savings account with interest and minimum balance
- Current account with overdraft facility

---

##  Concepts Used
- Classes and Objects  
- Inheritance  
- Method Overriding  
- Encapsulation
